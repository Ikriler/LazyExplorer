Desktop Automation

Last Updated: October 21, 2017
Beware of bugs!
The Wildfire Helper is fairly new and you may encounter some bugs, particularly during installation. If you run into any, please let us know at support@wildfire.ai.

As of Version 1.3.0, Wildfire can control the desktop system outside the context of the browser. This means your control is no longer limited to webpages and you may now manipulate the system. To use these features, you are required to install the Wildfire Helper application.

Note that events cannot yet be recorded outside of the browser context, so to use this functionality you may need to manually construct your events in the Workflow Editor. Currently, Google Chrome is the only supported browser for Desktop Automation. Firefox and Opera support may be added in the future.


Windows
Requires Windows 7 or above

Download the installer here.


Mac
Requires El Capitan (10.11) or above

Download the installer here.


Once installed, wait approximately one minute then verify the helper is working by navigating to the About section. Updates to the application happen automatically in the background.


Uninstalling

If you would like to uninstall the Wildfire Helper, follow the below instructions:


Windows

Open Windows Explorer and type or paste %LOCALAPPDATA% into the address bar. Right-click on the Wildfire folder and select the Delete option. When prompted, confirm that you wish to delete the folder.


Mac

Open Finder and navigate to Applications. Locate the Wildfire Helper icon, right-click on it, then select the Move To Trash option.
